# JavaScript30
## 目的
Javascript30是一个线上的教学课程。
利用三十天的时间每天实现一个简单的原生JavaScript作品，不能使用任何操作库。

## GitHub
项目是在GitHub上开源的，官方还给出了一个教程网站，里面有所有的视频教程。
[GitHub](https://github.com/wesbos/JavaScript30)
[教程网站](https://javascript30.com/)
